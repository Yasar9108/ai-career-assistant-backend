# AI Career Assistant Backend

AI-powered Career Assistant Backend built with Node.js, Express.js, MongoDB, and Gemini AI.

## Features

* JWT Authentication & Authorization
* Resume Analysis using Gemini AI
* Skill Gap Analysis
* Interview Question Generation
* Interview Answer Evaluation
* PDF Resume Upload & Parsing
* MongoDB Integration

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Gemini AI
* JWT
* Multer
* PDF-Parse

## API Modules

### Authentication

* Register User
* Login User

### Resume Analysis

* Resume Review
* ATS Optimization Suggestions

### Skill Gap Analysis

* Domain Match Percentage
* Missing Skills Detection
* Learning Roadmap

### Interview Preparation

* Generate Interview Questions
* Evaluate Single Answer
* Evaluate Multiple Answers
